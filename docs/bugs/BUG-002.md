- **Prioridade:** [ ] Urgente [x] Alta [ ] Normal [ ] Baixa

## Descrição
Testes Selenium falhavam com erro "This version of ChromeDriver only supports Chrome version 119" enquanto o Chrome instalado era versão 142.

## Passos para Reproduzir
1. Executar `npm test`
2. Observar falha em todos os testes Selenium
3. Verificar mensagem de incompatibilidade de versão

## Resultado Esperado
Testes Selenium deveriam executar normalmente com o Chrome instalado.

## Resultado Obtido
```
SessionNotCreatedError: session not created: This version of ChromeDriver only supports Chrome version 119
Current browser version is 142.0.7444.59
```

## Ambiente
- **Sistema Operacional:** Ubuntu 24 (WSL)
- **Navegador:** Google Chrome 142.0.7444.59
- **ChromeDriver:** 119.0.1
- **Selenium WebDriver:** 4.15.0

## Análise Técnica
Versão do pacote npm `chromedriver` estava desatualizada em relação à versão do Chrome instalada no sistema. ChromeDriver precisa ser compatível com a versão major do Chrome.

## Solução Implementada
```bash
# Desinstalar versão antiga
npm uninstall chromedriver

# Instalar versão mais recente
npm install --save-dev chromedriver@latest

# Limpar cache
npm cache clean --force
```

## Testes Realizados
- ✅ Execução completa da suite de testes

- ✅ TC001 a TC010 validados

## Resolução
Bug resolvido através da atualização do ChromeDriver para versão compatível com Chrome 142.
